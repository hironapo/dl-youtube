## 画面構成

アプリは大きく3つのエリアに分かれています。

```
┌──────────────────┬────────────────────────────────────────────┐
│                  │                                            │
│   サイドバー      │        メインエリア                         │
│                  │                                            │
│  ┌────────────┐  │  ┌──────────────────────────────────────┐  │
│  │ ナビ       │  │  │   YouTubeプレイヤー                   │  │
│  │ [動画]     │  │  │                                      │  │
│  │ [ボキャブ] │  │  └──────────────────────────────────────┘  │
│  └────────────┘  │                                            │
│                  │  ┌──────────────────────────────────────┐  │
│  ┌────────────┐  │  │ タブエリア                            │  │
│  │ 動画ツリー  │  │  │ [字幕][フレーズ][解説][コメント]     │  │
│  │ (日付別)   │  │  │                                      │  │
│  │            │  │  │   タブコンテンツ                     │  │
│  │            │  │  │                                      │  │
│  └────────────┘  │  └──────────────────────────────────────┘  │
│                  │                                            │
│  ┌────────────┐  │                                            │
│  │ DL入力欄   │  │                                            │
│  │ [URL][DL]  │  │                                            │
│  └────────────┘  │                                            │
└──────────────────┴────────────────────────────────────────────┘
```

### サイドバー

- **ナビゲーション**: 「動画ライブラリ」と「ボキャブラリー」の2つのメイン画面を切り替える
- **動画ツリー**: 日付別に折りたたみ式で動画が一覧表示される。クリックで動画を選択
- **DL入力欄**: YouTube URLを入力してダウンロード開始する
- **モデル選択**: 右上のドロップダウンでLLMモデルを切り替える
- **ヘッダー検索**: 検索ボックスに入力すると動画・フレーズをリアルタイム検索できる

### プレイヤーエリア

YouTube IFrame APIによる埋め込みプレイヤー。字幕クリックで該当箇所にシークできる。

### タブエリア

動画選択後に以下の4タブが利用できる。

| タブ | 内容 |
|------|------|
| 字幕 | 英語（左列）・日本語（右列）の2列字幕表示 |
| フレーズ | LLMが抽出した重要フレーズ一覧 |
| 解説 | 選択フレーズの語源・ゴロ合わせ等の詳細解説 |
| コメント | 動画のYouTubeコメント一覧 |

## 動画のダウンロードと登録

### YouTube URLからダウンロードする

1. サイドバー下部のURL入力欄にYouTube URLを貼り付ける
2. LLMモデルをドロップダウンで選択する（デフォルト: claude-3-haiku）
3. **「DL」ボタン**をクリックする
4. ログパネルがポップアップし、進捗がリアルタイムで表示される

ダウンロード完了後、以下の処理が自動的に行われる。

- 動画ファイル（`.mp4`）の保存
- 英語字幕（`.en.srt`）の取得
- LLMによるフレーズ抽出（`.md`）の生成
- データベースへの登録

ダウンロードが完了すると動画ツリーに新しい動画が追加される。

### 対応URL形式

- `https://www.youtube.com/watch?v=XXXXX`
- `https://youtu.be/XXXXX`
- プレイリストURL（ただし最初の動画のみ処理）

## 字幕の見方と操作

### 2列表示

「字幕」タブを開くと、英語字幕（左列）と日本語字幕（右列）が2列で表示される。

```
┌──────────────────────────────┬──────────────────────────────┐
│ 英語字幕                      │ 日本語字幕                    │
├──────────────────────────────┼──────────────────────────────┤
│ 00:00:01                     │ 00:00:01                     │
│ Never gonna give you up      │ 君を手放すことは絶対しない     │
├──────────────────────────────┼──────────────────────────────┤
│ 00:00:04                     │ 00:00:04                     │
│ Never gonna let you down     │ 君を失望させることは絶対しない │
└──────────────────────────────┴──────────────────────────────┘
```

### 字幕クリックの挙動

| 操作 | 動作 |
|------|------|
| 字幕行を1回クリック | プレイヤーが該当タイムスタンプにシーク（静止状態） |
| 同じ字幕行をもう1回クリック | その位置から動画を再生開始 |
| 別の字幕行をクリック | 新しいタイムスタンプにシーク |

### スライディングウィンドウ重複除去

英語字幕と日本語字幕はそれぞれ重複する内容を自動的に除去して表示する。字幕ファイルに同一テキストが連続する場合でも1行だけ表示されるため、読みやすくなっている。

### 字幕が表示されない場合

英語字幕または日本語字幕がない場合は空欄になる。字幕取得ボタン（後述）で個別に取得できる。

## フレーズ解説の使い方

### フレーズ一覧の確認

「フレーズ」タブを開くと、LLMが動画字幕から抽出した重要フレーズが一覧表示される。各フレーズには以下の情報が表示される。

- 英語フレーズ
- 日本語訳
- 文法難易度バッジ（💡 / 💡💡 / 💡💡💡）

### フレーズの解説を見る

フレーズをクリックすると「解説」タブに切り替わり、以下の詳細情報が表示される。

| 項目 | 内容 |
|------|------|
| 定義 | 英語の辞書的定義 |
| 日本語訳 | 意味・訳 |
| 語源 | ラテン語・ギリシャ語・古英語などの語源解説 |
| 言語学的考察 | 句動詞・コロケーション・文法的特徴など |
| 覚え方ストーリー | 記憶に残るイメージストーリー（日本語） |
| ゴロ合わせ | 日本語のゴロ合わせ |
| 例文 | 使用例文（英語）と日本語訳 |
| 関連語 | 類義語・反義語・関連語のリスト |
| 試験メモ | 英検1級・TOEICなどでの出題傾向 |

### 関連語から連鎖的に調べる

解説パネルに表示される「関連語」のタグをクリックすると、そのフレーズの解説がすぐに生成される。関連語から関連語へと連鎖的に語彙を広げていける。

### フレーズをボキャブラリーに登録する

解説パネルの**「★登録」ボタン**をクリックすると、フレーズがDBに保存される。登録したフレーズは「ボキャブラリー」画面で管理できる。

## ボキャブラリー管理

サイドバーの「ボキャブラリー」ナビをクリックするとボキャブラリー管理画面に切り替わる。

### フィルター機能

画面上部のフィルターボタンで表示するフレーズを絞り込める。

| フィルター | 説明 |
|-----------|------|
| すべて | 全登録フレーズを表示 |
| ★お気に入り | `is_top = true` のフレーズのみ表示 |
| 📹この動画 | 現在選択中の動画に紐づくフレーズのみ表示 |

### タグによる絞り込み

フレーズにはタグを付けて管理できる。タグフィルターをクリックすると該当タグのフレーズだけが表示される。

### お気に入りへの登録

フレーズカードの星アイコンをクリックするとお気に入り（`is_top`）フラグが切り替わる。重要度の高い単語を優先的に復習できる。

### フレーズカードの情報

ボキャブラリー画面のカードには以下が表示される。

- 英語フレーズ・日本語訳
- タグ
- 関連動画へのリンク
- 語源サマリー

## コメント取得と単語調べ

### コメントの取得

「コメント」タブで以下のいずれかの操作で取得できる。

- **「コメント取得」ボタン**: 現在の動画のコメントを取得
- **「全動画コメント取得」**: 全登録動画のコメントを一括取得（時間がかかる）

取得されたコメントはいいね順などで表示される。

### コメント内の単語を調べる

コメントテキスト内の英単語を選択（ドラッグ）してクリックすると、そのフレーズの解説が生成される。気になる表現をその場で即座に調べられる。

### コメントのフレーズを登録する

調べたフレーズが重要だと感じたら「★登録」ボタンでボキャブラリーに追加できる。コメントから学んだ表現も一元管理できる。

## ネットワーク表示

### ネットワークとは

登録フレーズ間の意味的な関連（類義・反義・関連語など）をforce-directedグラフで可視化する機能。単語の繋がりを視覚的に把握できる。

### ネットワークの表示方法

1. ボキャブラリー画面でネットワークタブに切り替える
2. ノード（円）がフレーズ、線がフレーズ間の関連を表す
3. ノードをドラッグして配置を調整できる
4. ノードをクリックするとフレーズの解説が表示される

### ネットワークの自動構築

**「ネットワーク構築」ボタン**をクリックすると、LLMが登録済みフレーズ間の関連を自動解析してリンクを生成する（`POST /api/network/build`）。フレーズ数が多いほど時間がかかる。

### ネットワークの見方

| 表示要素 | 意味 |
|---------|------|
| ノードのサイズ | フレーズの登録頻度・重要度 |
| ノードの色 | フレーズのカテゴリ・グループ |
| 線の太さ | 関連の強さ（weight） |
| 線の種類 | 関連タイプ（synonym/antonym/related） |

## LLMモデルの選択

### モデルの切り替え

画面右上のドロップダウンメニューからLLMモデルを選択できる。選択したモデルはフレーズ解説の生成・ダウンロード時のフレーズ抽出の両方に使用される。

### 各モデルの特徴とコスト

| モデル | 特徴 | コスト | 推奨用途 |
|--------|------|--------|---------|
| **claude-3-haiku** | 高速・低コスト・品質良好 | 低 | 日常的なフレーズ解説（デフォルト） |
| **claude-sonnet-4-6** | 高品質・詳細な解説 | 中〜高 | 複雑な語源・言語学的考察が必要な時 |
| **deepseek-chat** | コスト効率が高い | 低 | 大量処理・コスト節約時 |
| **gemini-2.0-flash-exp** | 無料枠あり | 無料〜低 | API費用を抑えたい時 |
| **llama-3.3-70b-instruct** | 無料枠あり・オープンソース | 無料〜低 | コスト0で試したい時 |
| **gpt-4o-mini** | バランス型・安定した品質 | 低〜中 | OpenAIの品質を試したい時 |

### 推奨設定

- **通常学習**: `claude-3-haiku`（コストと品質のバランスが最も良い）
- **深い語源調査**: `claude-sonnet-4-6`（詳細な言語学的解説を期待する場合）
- **無料で使いたい**: `gemini-2.0-flash-exp` または `llama-3.3-70b-instruct`

## 字幕の個別取得ボタン

動画メタデータが登録された後でも、字幕を個別に取得・更新できる。

### 英語字幕の取得

「字幕」タブの**「EN字幕取得」ボタン**をクリックすると、`POST /api/video/<video_id>/fetch_en_sub` が呼び出され、yt-dlpで英語字幕を取得・DB保存する。

**使う場面:**
- DL時に自動字幕取得が失敗した場合
- 後から英語字幕を更新したい場合
- 手動字幕と自動生成字幕を切り替えたい場合

### 日本語字幕の取得

「字幕」タブの**「JA字幕取得」ボタン**をクリックすると、`POST /api/video/<video_id>/fetch_ja_sub` が呼び出され、日本語字幕を取得する。

**使う場面:**
- DL後に日本語字幕を追加取得したい場合
- 日本語字幕が最初から存在しなかった動画に後で字幕が追加された場合

### 全動画の日本語字幕一括取得

設定メニューの**「全JA字幕一括取得」ボタン**を使うと `POST /api/fetch_all_ja_subs` が呼び出され、登録済み全動画の日本語字幕をまとめて取得する。動画数が多い場合は時間がかかる。

## キーボードショートカット

アプリ内で使えるキーボードショートカット一覧。

### プレイヤー操作

| キー | 動作 |
|------|------|
| `Space` | 動画の再生/一時停止 |
| `←` | 5秒巻き戻し |
| `→` | 5秒早送り |
| `↑` | 音量アップ |
| `↓` | 音量ダウン |

### タブ切り替え

| キー | 動作 |
|------|------|
| `1` | 字幕タブを表示 |
| `2` | フレーズタブを表示 |
| `3` | 解説タブを表示 |
| `4` | コメントタブを表示 |

### フレーズ操作

| キー | 動作 |
|------|------|
| `Enter` | 選択中フレーズの解説を生成 |
| `S` または `s` | 選択中フレーズをボキャブラリーに登録 |

| キー | 動作 |
|------|------|
| `/` または `Ctrl + K` | ヘッダー検索ボックスにフォーカス |
| `Esc`（検索中） | 検索をクリアしてフォーカスを外す |

| キー | 動作 |
|------|------|
| `N` または `n` | 次の動画へ移動 |
| `P` または `p` | 前の動画へ移動 |

## トラブルシューティング

### 動画がダウンロードできない

- yt-dlpが最新版かどうかを確認する: `yt-dlp --update`
- 動画が年齢制限・地域制限を受けていないか確認する
- Cookiesファイルの設定が必要な場合がある

### 字幕が取得できない

- 対象動画に字幕が存在しない場合がある
- 自動生成字幕のみの動画は取得できない場合がある
- yt-dlpを最新版に更新する

### LLM解説が生成されない

- `.env` の `OPENROUTER_API_KEY` が正しく設定されているか確認する
- OpenRouterの残高・レート制限を確認する
- 無料モデル（gemini-2.0-flash-expまたはllama-3.3-70b-instruct）に切り替えてみる

### DBが壊れた場合

- `~/youtube_phrases.db` のバックアップがあれば復元する
- バックアップがない場合、ファイルを削除するとアプリ起動時に新規作成される（データは失われる）

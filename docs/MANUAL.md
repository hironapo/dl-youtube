## 画面構成

アプリは大きく3つのエリアに分かれています。

```
┌──────────────────┬──────────────────────┬──────────────────┐
│                  │                      │                  │
│   サイドバー      │  YouTubeプレイヤー    │  右パネル(タブ)  │
│                  │                      │                  │
│  [動画ライブラリ] │  ┌────────────────┐  │  [字幕]          │
│  [ボキャブラリー] │  │  YouTube       │  │  [フレーズ]      │
│                  │  │  Embed         │  │  [🔍解説]         │
│  動画ツリー       │  └────────────────┘  │  [📷学習]         │
│  (日付別)        │                      │  [💬コメント]     │
│                  │  現在字幕表示         │                  │
│                  │  ━━━━━━━━━━━━━━━━    │  タブコンテンツ   │
│                  │  字幕リスト           │                  │
│                  │  (EN | JA | 📄全文)  │                  │
└──────────────────┴──────────────────────┴──────────────────┘
```

### サイドバー

- **ナビゲーション**: 「動画ライブラリ」と「ボキャブラリー」の2つのメイン画面を切り替える
- **動画ツリー**: 日付別に折りたたみ式で動画が一覧表示される。クリックで動画を選択
- **ヘッダー検索**: 検索ボックス（Ctrl+K）に入力すると動画・フレーズをリアルタイム検索

### プレイヤーエリア

YouTube IFrame APIによる埋め込みプレイヤー。字幕クリックで該当箇所にシークできる。

### 右パネル（タブ）

動画選択後に以下の5タブが利用できる。

| タブ | 内容 |
|------|------|
| 字幕 | 英語（左列）・日本語（右列）の2列字幕表示 |
| フレーズ | LLMが抽出した重要フレーズ一覧 |
| 🔍 解説 | 選択フレーズの語源・ゴロ合わせ等の詳細解説 |
| 📷 学習 | スクリーンショット＋センテンス精聴モード |
| 💬 コメント | 動画のYouTubeコメント一覧 |

## 動画のダウンロードと登録

### YouTube URLからダウンロードする

1. ヘッダーの**「📥 DL・登録」ボタン**をクリックしてモーダルを開く
2. テキストエリアにYouTube URLを貼り付ける（複数行で複数URL入力可）
3. LLMモデルをドロップダウンで選択する（デフォルト: claude-3-haiku）
4. **「DL開始」ボタン**をクリックする
5. ログパネルに進捗がリアルタイムで表示される

ダウンロード完了後、以下が自動実行される。

- 動画ファイル（`.mp4`）の保存
- 英語字幕（`.en.srt`）の取得
- LLMによるフレーズ抽出（`_phrases.md`）の生成
- データベースへの登録

### 対応URL形式

- `https://www.youtube.com/watch?v=XXXXX`
- `https://youtu.be/XXXXX`

## 字幕の見方と操作

### 2列表示

「字幕」タブを開くと、英語字幕（左列）と日本語字幕（右列）が2列で表示される。

```
時刻  │ English                    │ 日本語               │ 難易度
──────┼────────────────────────────┼──────────────────────┼──────
0:01  │ Never gonna give you up    │ 君を手放すことはない  │ 💡
0:04  │ Never gonna let you down   │ 君を失望させない      │
```

### 字幕クリックの挙動

| 操作 | 動作 |
|------|------|
| 字幕行を1回クリック | プレイヤーが該当タイムスタンプにシーク（静止） |
| 同じ字幕行をもう1回クリック | その位置から動画を再生開始 |
| `.+` ボタン | フレーズ登録モーダルを開く |

### 単語クリックとテキスト選択

| 操作 | 動作 |
|------|------|
| 英語の単語を1クリック | 単語ポップアップを表示（🔍解説 / 📌登録） |
| 英語テキストをマウスでドラッグ選択 | 選択範囲のポップアップを表示（🔍解説 / 📌登録） |

### 難易度バッジ

| バッジ | レベル |
|--------|--------|
| 💡 | 初級〜中級（基本構文・一般語彙） |
| 💡💡 | 中上級（やや複雑な構文・準高度語彙） |
| 💡💡💡 | 上級・英検1級レベル（複雑構文・高度語彙） |

## A-B区間ループ

字幕リストの上部にループコントロールバーがある。

| ボタン | 動作 |
|--------|------|
| 🔁 全体 | 動画全体を繰り返し再生する |
| A 始点 | 現在時刻をループ開始点（A点）に設定 |
| B 終点 | 現在時刻をループ終了点（B点）に設定 |
| ✕ 解除 | ループを解除する |

A点・B点を設定するとその区間を繰り返し再生できる。字幕行をクリックしてA点に移動してからA/Bを設定すると便利。

## フレーズ解説の使い方

### 解説の表示

以下の操作で「🔍 解説」タブに詳細解説が表示される。

- 字幕の単語をクリック
- 字幕テキストをドラッグ選択 → ポップアップから「🔍 解説」
- フレーズタブの「解説」ボタン
- 現在表示中の字幕の「🔍 解説」ボタン（字幕エリア右上）

解説には以下が含まれる。

| 項目 | 内容 |
|------|------|
| 意味・説明 | 日本語による詳細な意味 |
| 使い方・ニュアンス | 語法・コロケーション |
| 例文 | 英検1級レベルの例文＋日本語訳 |
| 🌱 語源 | ラテン語・ギリシャ語・古英語等の語根分析 |
| 🔬 言語学的考察 | 音象徴・認知言語学的メタファー・形態素分析 |
| 📖 覚え方ストーリー | 記憶に残るエピソード（日本語） |
| 🎵 ゴロ合わせ | 音・形・意味を結びつける記憶術 |
| 関連語 | 英検1級以上レベルの類語・対義語・派生語 |
| 英検1級メモ | 試験での出題傾向・注意点 |

### 解説のDBキャッシュ

解説は初回取得時にDBに自動保存される。同じフレーズの2回目以降はDBから即座に返却されるため、API呼び出しが発生しない。

### 解説一括取得

ボキャブラリータブの**「🔄 解説一括取得」ボタン**を使うと、登録済みフレーズで解説未取得のものをバックグラウンドで一括プリフェッチできる。

## 全文スクリプトダウンロード

字幕ヘッダーの**「📄 全文」ボタン**をクリックすると全文スクリプトモーダルが開く。

### 内容

- 動画タイトル・チャンネル名・再生時間
- 全センテンスのEN+JA対応（1センテンスずつ）
- 登録済みフレーズの解説（マッチしたセンテンスに付加）

### フォーマット例

```markdown
# 動画タイトル
📺 チャンネル名 | ⏱ 15:23
---

### ▶ [0:00:01]
🇺🇸 **You are not ready for how good that is.**
🇯🇵 あなたはそれがどれほど良いか準備できていない。

  📖 **all that good**
  > 意味: 非常に優秀な...
  > 語源: ...
  > 覚え方: ...
```

### 操作

| 操作 | 動作 |
|------|------|
| 「解説含む」チェックボックス | ON/OFFでリアルタイム切り替え |
| 「⬇ DL (.md)」ボタン | Markdownファイルとして保存 |
| 背景クリック or ESC | モーダルを閉じる |

## 学習モード

右パネルの**「📷 学習」タブ**で精聴・精読練習ができる。

### 機能

- 現在のセンテンスのCanvasスクリーンショット（動画の静止画）
- センテンスのEN/JA表示
- **「🔍 英検1級解説」ボタン**でそのセンテンスの解説を表示
- **← →** キーで前後センテンスに移動

### 操作

| キー / ボタン | 動作 |
|------|------|
| `←` / `→` | 前後のセンテンスに移動 |
| `Enter` | 現在センテンスの解説を生成 |
| 🔍 英検1級解説 | OpenRouterで詳細解説を生成 |

## ボキャブラリー管理

サイドバーの「ボキャブラリー」をクリックするとボキャブラリー管理画面に切り替わる。

### フィルターボタン

| フィルター | 説明 |
|-----------|------|
| すべて | 全登録フレーズを表示 |
| ★お気に入り | `is_top = 1` のフレーズのみ |
| 📹この動画 | 現在選択中の動画のフレーズのみ |
| 📖 単語帳 | 単語（1語）のみ・フリップカード表示 |
| 📝 フレーズ帳 | 複数語フレーズのみ |
| 🎯 英検1級 | 英検1級レベルのフレーズのみ |

### グループ表示

| ボタン | 動作 |
|--------|------|
| タグ別 | タグでグループ分けして表示 |
| 📅 日付別 | 登録日付（降順）でグループ分けして表示 |

### 単語帳モード（フリップカード）

「📖 単語帳」フィルターを選択するとフリップカード表示に切り替わる。

- **表面**: 英単語 + 語源ヒント
- **裏面**: 日本語訳 + ゴロ合わせ
- カードをクリックして表裏を切り替える

### フレーズカード操作

| ボタン | 動作 |
|--------|------|
| 解説 | 英検1級解説モーダルを開く |
| ★ | お気に入りをトグル |
| 削除 | フレーズを削除 |

## ネットワーク表示

ボキャブラリー画面の**「🌐 ネットワーク」ボタン**でフレーズ間の関連をグラフ表示する。

- ノード（円）がフレーズ、線がフレーズ間の関連を表す

## LLMモデルの選択

ヘッダー右のドロップダウンメニューからLLMモデルを選択できる。

| モデル | 特徴 | コスト |
|--------|------|--------|
| **claude-3-haiku** | 高速・低コスト・品質良好 | 低 |
| **claude-sonnet-4-6** | 高品質・詳細な解説 | 中〜高 |
| **deepseek-chat** | コスト効率が高い | 低 |
| **gemini-2.0-flash-exp** | 無料枠あり | 無料〜低 |
| **llama-3.3-70b-instruct** | 無料枠・オープンソース | 無料〜低 |
| **gpt-4o-mini** | バランス型・安定 | 低〜中 |

**推奨:**
- 通常学習: `claude-3-haiku`
- 深い語源調査: `claude-sonnet-4-6`
- 無料利用: `gemini-2.0-flash-exp` または `llama-3.3-70b-instruct`

## トラブルシューティング

### 動画がダウンロードできない

- yt-dlpを最新版に更新する: `yt-dlp --update`
- 動画が年齢制限・地域制限を受けていないか確認する

### 字幕が表示されない

- 対象動画に字幕が存在しない場合がある
- 「EN字幕取得」「JA字幕取得」ボタンで個別に取得できる
- yt-dlpを最新版に更新する

### LLM解説が生成されない

- `.env` の `OPENROUTER_API_KEY` が正しいか確認する
- OpenRouterの残高・レート制限を確認する
- 無料モデル（gemini-2.0-flash-exp または llama-3.3-70b-instruct）に切り替えてみる

### 解説がJSONのまま表示される

- 解説の取得が失敗している可能性がある
- 再度「解説」ボタンをクリックすると再取得できる

### DBが壊れた場合

- `~/youtube_phrases.db` を削除するとアプリ起動時に新規作成される（データは失われる）
